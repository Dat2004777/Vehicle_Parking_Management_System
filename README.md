## 🛠 Tech Stack
* **Backend:** Java Servlet, JSP, JSTL.
* **Database:** Microsoft SQL Server.
* **Frontend:** Bootstrap 5, Javascript (ES6), CSS3 (Custom Animations).
* **Security:** BCrypt Password Hashing, Role-based Authorization Filter.

## 🚀 Key Features
- [x] **Real-time Check-in/out:** Kiểm soát thẻ RFID và nhận diện biển số.
- [x] **Smart Toast Component:** Thông báo trượt tích hợp Progress bar (Custom Taglib).
- [x] **Subscription Management:** Đăng ký và gia hạn vé tháng tự động.
- [x] **Authorization Filter:** Chặn truy cập trái phép bằng Filter & Role Prefix.
- [x] **Responsive Design:** Giao diện tối ưu cho cả Desktop và Mobile.
