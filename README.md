> Một hệ thống quản lý bãi đỗ xe thông minh hỗ trợ kiểm soát xe ra vào, quản lý vé tháng và tính phí tự động, được xây dựng trên nền tảng Java Web MVC.

---

## 🛠 Tech Stack
* **Backend:** Java Servlet, JSP, JSTL.
* **Database:** Microsoft SQL Server.
* **Frontend:** Bootstrap 5, Javascript (ES6), CSS3 (Custom Animations).
* **Security:** BCrypt Password Hashing, Role-based Authorization Filter.

## 🚀 Key Features
- [x] **Check-in/out:** Quy trình kiểm soát thẻ RFID và biển số xe thời gian thực.
- [x] **Smart Toast:** Hệ thống thông báo trượt (Toast) tích hợp thanh Progress bar.
- [x] **Subscription:** Quản lý và gia hạn vé tháng linh hoạt.
- [x] **Authorization:** Phân quyền chặt chẽ giữa Admin, Manager và Staff.
